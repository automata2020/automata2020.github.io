DEBUGGING
Finding errors in given code.

• Number of rounds : 3

ROUND 1:
Each participant will be handed 3 c++ codes in which he/she has to find errors in 25 minutes.

ROUND 2:
Each participant will be handed 1 java code in which he/she has to find errors in 10 minutes.

ROUND 3:
Each participant will be given some situations to break which occur where ever wrong code (like infinity loop) which he/she must exit without closing app in 5 minutes per situation.


• In case of a tie time of submission will be considered.
• Maximum number of participants per team is 2.
